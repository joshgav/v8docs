18  // Evaluate a piece of JavaScript in the context of a stack frame for
19  // debugging. Things that need special attention are:

21  // values need to be written back to the stack afterwards.

28  // This class builds a context chain for evaluation of expressions
29  // in debugger.
30  // The scope chain leading up to a breakpoint where evaluation occurs
31  // looks like:
32  // - [a mix of with, catch and block scopes]
33  // - [function stack + context]
34  // - [outer context]
35  // The builder materializes all stack variables into properties of objects;
36  // the expression is then evaluated as if it is inside a series of 'with'
37  // statements using those objects. To this end, the builder builds a new
38  // context chain, based on a scope chain:
39  // - every With and Catch scope begets a cloned context
40  // - Block scope begets one or two contexts:
41  // - if a block has context-allocated varaibles, its context is cloned
42  // - stack locals are materizalized as a With context
43  // - Local scope begets a With context for materizalized locals, chained to
44  // original function context. Original function context is the end of
45  // the chain.
46  class ContextBuilder {
47  public:
48  ContextBuilder(Isolate* isolate, JavaScriptFrame* frame,
49  int inlined_jsframe_index);
50 
51  void UpdateValues();
52 
53  Handle<Context> evaluation_context() const { return evaluation_context_; }
54  Handle<SharedFunctionInfo> outer_info() const { return outer_info_; }

56  private:
