const int v8::internal::LiveEdit::kFramePaddingInitialSize = 1
static

Defines layout of a stack frame that supports padding. This is a regular internal frame that has a flexible stack structure. LiveEdit can shift its lower part up the stack, taking up the 'padding' space when additional stack memory is required. Such frame is expected immediately above the topmost JavaScript frame.
